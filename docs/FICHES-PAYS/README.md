## 🚨 Risk Management

### Risk Escalation Matrix

| Severity | Examples | Escalation |
|----------|----------|-----------|
| **CRITICAL** | Security incident (RDC), major government withdrawal, > €100K budget overrun | CEO + board (within 24h) |
| **HIGH** | Timeline slip > 4 weeks, > €50K revenue miss, major technical issue | PM + exec team (within 48h) |
| **MEDIUM** | Timeline slip 1-3 weeks, > 20% revenue miss, stakeholder friction | PM + regional lead (within 1 week) |
| **LOW** | Minor delays, staffing changes, local issue | PM (track, report monthly) |

### RDC Special Protocol (Very High Risk)

**Applicable ONLY if RDC project proceeds:**

2. Monthly risk reassessment (abort trigger if Kivu escalates)

## ❓ FAQs

### Q: Which country should we start with?
**A:** MAROC (highest confidence, existing relationship, go-live June 2026). If you have WB relationship, add SENEGAL in parallel.

### Q: What if a country doesn't hit revenue targets?
**A:** Likely; Year 1 is establishment. Track actual vs. projection, adjust Year 2 forecast by October 2026. Discuss mitigation (longer sales cycle, lower adoption) rather than abandoning.

### Q: Can we change the order (e.g., start MALAWI before MAROC)?
**A:** Technically yes, but MAROC is reference case. Recommend parallel deployments (MAROC + SENEGAL in Q2-Q3, others in Q3-Q4) rather than sequential.

### Q: What's the biggest risk?
**A:** RDC (security, infrastructure, complexity). MAROC/MALAWI lowest risk. SENEGAL/BENIN medium-manageable.

### Q: When do we know if a country is "working"?
**A:** By Month 4-6 post-go-live. Watch: adoption rate (% of target users onboarded), transaction volume (trajectory toward Year 1 target), revenue (platform fees collected), NPS (stakeholder satisfaction).

### Q: Can we reuse MAROC code/contracts for other countries?
**A:** ~70% yes (core SafeLand stack is portable). 30% customization: legal templates, tax logic, customary law rules, language/UI. Budget €20-30K per country for customization.

### Q: Who owns what if there are partnership disputes?
**A:** Defined in each country MOU (government, partner, SafeLand). MAROC model is template; others adjust per local context.
