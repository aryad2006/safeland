# Proposition de Pilote SafeLand
## Deploiement sur 3 Conservations Foncieres

**Emetteur :** TachyDigital SARL
**Destinataires :** ANCFCC, MTNRA
**Date :** Mars 2026

---

## 1. Resume

Nous proposons un pilote de 6 mois pour deployer la plateforme SafeLand sur 3 conservations foncieres (Casablanca, Rabat, Marrakech), tokenisant 1 000 titres fonciers et formant 20 agents.

---

## 2. Objectifs du Pilote

| Objectif | KPI | Seuil de Succes |
|----------|-----|-----------------|
| Tokeniser des titres reels | Nombre de NFTs crees | >= 1 000 |
| Valider le workflow complet | Transactions Escrow completees | >= 50 |
| Mesurer la reduction de fraude | Fraudes detectees/bloquees | Baseline etablie |
| Tester la fiscalite at-the-source | Transactions avec split DGI/ANCFCC | 100% des ventes |
| Former les agents | Agents operationnels | >= 20 |
| Evaluer la satisfaction | Score satisfaction agents | >= 80% |

## 7. Risques et Mitigation

| Risque | Probabilite | Impact | Mitigation |
|--------|-------------|--------|------------|
| Resistance des agents | Moyenne | Haute | Formation progressive, accompagnement terrain |
| Qualite donnees sources | Haute | Moyenne | Nettoyage prealable, validation croisee |
| Indisponibilite reseau | Faible | Haute | Mode degrade, cache local |
| Bug technique | Moyenne | Moyenne | Equipe dedicee, hotfix < 48h |
| Changement de priorite ANCFCC | Moyenne | Haute | Convention signee, comite de suivi mensuel |

---

## 8. Criteres de Succes

Le pilote sera considere comme un succes si :

1. >= 1 000 titres tokenises sans erreur
2. >= 50 transactions Escrow completees avec split fiscal
3. 0 incident de securite critique
4. Satisfaction agents >= 80%
5. Disponibilite plateforme >= 99%
6. Rapport d'evaluation positif
